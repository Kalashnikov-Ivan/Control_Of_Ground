Игра "Контроль земли"

I. Название игры
	Это название было выбрано с отсылкой на главную механику игры -
	способность создавать и трансформировать игровое поле в процессе
	самой игры. Главная особенность в том, что манипуляции с окружением
	можно использовать как оружие в нападении или защите.
	Главная цель игры: одержание победы над соперником в том или ином
	соревновательном режиме. Пока что он один: дуэль.

III. Экраны
	1. Главное меню (MainMenu)
		При запуске программы входной точной будет являться главное меню,
		в котором будут содержаться три кнопки:
												START
												SETTINGS
												ExIT
		Более подробное их описание будет дана далее.

	->Элементы (объекты):
			1. START - запуск игры, переход на экран выбора карты, параметров
			   персонажа (умения, оружие, имя). В основной игровой цикл
			2. SETTINGS - переход на экран настроек (расширение экрана,
			   настройка звука, управления). В цикл экрана SETTINGS
			3. ExIT - выход из игры. Событие закрытия окна
			
	->Проблемы:
			1. Реализация обработки событий перемещения курсора мыши,
			   чтобы программа ослеживала координаты положения курсора и
			   реагировала если будет нажата левая кнопка мыши (ЛКМ) активируя
			   переход на указанный экран.
			   	а) Возможнось изменение текстуры или появление какого-либо
			   	   эффекта, если курсор был наведён на элемент.
			2. Реализация каждого элемента не как некой абстрактной области
			   координат, при нажатии на которую будет происходить какое-либо
			   событие, а как отдельного объекта.
			   Я думаю, сделать это можно будет через sf::RectangleShape
			3. Обновление экрана по случаю какого-либо события для избежания
			   ненужных итераций основного цикла.

	2. Настройки (SETTINGS)
		Будет выводиться окошко для выбора предложенного (стандартного)
		разрешения экрана, выбор режима окна, настройки громкости и вообще
		наличия звука, кнопок управления (вверх, вниз, вперёд, назад, атака 
		оружием 1, атака оружием 2, использование способности 1,2,3).
		Кнопку возврата в Главное меню - Back
		Кнопку применения настроек - Apply

			->Элементы (объекты):
			I. Настройки графики (Video settings):
				1. Разрешение экрана - настройка будет иметь две кнопки
				   переключения и одно окно отображения состояния:
													   				pos0-800х600 (default)
													   				pos1-1280х720
													   				pos2-1600х900
													   				pos3-1920х1080
					При нажатии на кнопки (вправо/влево) будет выбираться разрешение:
					вправо - позиция ++
					влево - позиция --

				2. Режим окна - будет представлять из себя кнопку-флажок (вкл/выкл):
												Нет квадратика = OFF-полноэкранный режим 
												Есть квадратик = ON-оконного режима с рамкой (default)

			II. Настройки звука (Audio):
				1. Громкость - панель отображения состояния громности в процентах (100% default)
				   и две кнопки изменения значений:
				   									вправо (+5%)
				   									влево (-5%)
				2. Громкость вкл/выкл (vol ON/OFF) - кнопка-флажок (вкл/выкл)
			III. Настройки управления (Control):
				Пока что оставим настройки управления по-умолчанию без возможности их изменения


	3. Подготовка к игре (PrePlayMenu)
		Будет содержаться информация об элементах управления персонажем, его способностях, оружии и характеристиках; игровом уровне, кнопка начать игры на данном уровне.

	4. Игровое поле (PlayGround)
		Самый главный игровой экран, в котором будет отображаться поле, игроки, NPC и всё остальное.
		В нём будет происходить главный игровой процесс.

		->Элементы (объекты):
		1. Спрайт игрока - будет представлять собой прямоугольник, который будет чуть равен или чуть
		   больше размеру одного блока карты.
